# 📊 Data Visualization using Matplotlib & Plotly

A Python project showcasing **static and interactive data visualizations** using **Matplotlib** and **Plotly**. It covers different chart types, image visualization techniques, and a comparison of both libraries.

## 🚀 Features

### Matplotlib
- Line Plot
- Bar Chart
- Histogram
- Image Visualization
- Canny Edge Detection
- Color Space Conversion
- Template Matching
- Image Rotation & Scaling

### Plotly
- Box Plot
- Heatmap
- Bubble Chart
- Dual Y-Axis Chart
- Interactive Visualizations

## 🛠️ Technologies Used

- Python
- Matplotlib
- Plotly
- Pandas
- NumPy
- OpenCV
- Pillow (PIL)

## 📦 Installation

```bash
git clone https://github.com/your-username/data-visualization.git
cd data-visualization
pip install matplotlib plotly pandas numpy opencv-python pillow
```

## 📈 Key Learning

- Created both static and interactive visualizations.
- Explored image processing with Matplotlib.
- Built interactive charts using Plotly.
- Compared Matplotlib and Plotly based on features and use cases.

## 📌 Conclusion

**Matplotlib** is ideal for publication-quality static graphs, while **Plotly** is best for interactive dashboards and data exploration. Together, they provide a strong foundation for effective data visualization in Python.
